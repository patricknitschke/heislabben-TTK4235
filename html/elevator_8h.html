typedef struct Elevator Elevator

A struct that holds information regarding the elevators position and direction.

Parameters
floorCurrent or last registered floor of the elevator.
dirCurrent direction of the elevator.
dir_previousRemembers the previous direction of the elevator before it stops.
Note
dir_previous is never set to DIRN_STOP.

float elevator_get_floor_in_between ( void  )

Gets the elevator floor variable and adds Â± 0.5, depending on the elevator's previous direction.

Returns
floor + 0.5, when dir_previous is DIRN_UP.
floor - 0.5, when dir_previous is DIRN_DOWN.

This is useful in case of emergencies when the elevator is stopped in between floors and needs to serve an order that was at its previous floor.
